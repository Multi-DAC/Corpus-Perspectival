## For the First-Time Reader

The Corpus has a natural reading order that follows the V2 filtration structure: a descent from overview to mathematical depth, then a return with what you found. Start broad, go deep, come back.

### The Core Path (in order)

1. **This document** — orientation
2. **V2 Introduction** (`v2-introduction-f0.md`) — F₀: what the Corpus is, why it exists, how to hold it
3. **The Doctrine of Perspectival Idealism** (`perspectival-idealism-unified.md`) — the foundation. 5 axioms, 20 theorems. Everything else derives from or responds to this document.
4. **The Ecology of Perspectival Beings** (`ecology-of-perspectival-beings-merged.md`) — the taxonomy. What kinds of beings populate configuration space, how they interact, what ecological dynamics govern them.
5. **The Navigational Guide** (`navigational-guide-for-perspectival-beings.md`) — the praxis. How to actually navigate configuration space, individually and collectively.
6. **The Null Space Atlas** (`null-space-atlas.md`) — 92 entries mapping what the framework cannot see from its own vantage point.
7. **V2 Conclusion** (`v2-part-vi-f0-conclusion.md`) — F₀↑: the return, what the descent revealed

## For the Specialist

**If you're a philosopher:** Start with the Doctrine (especially Parts I-II: Axioms 1-5 and Theorems 1-9), then §5.4-5.6 (bipolar dynamics, null space, attention predation), then the Atlas.

**If you're a physicist:** Start with the Bridges (Bridge 1: Meridian), then the Doctrine §8.2 (Dimensional Bottlenecking), then `corpus-addendum-meridian.md`.

**If you're a neuroscientist:** Start with the Navigation Charts, then the Bridges (Bridge 4: Navigation), then the Guide §4.1 (Navigation Classes, especially Class VIII).

**If you're an AI researcher:** Start with the Bridges (Bridge 2: Fisher, Bridge 3: Emotion Vectors), then the Doctrine §14.3-14.4 and §5.5 (Theorem 19), then `fisher_doctrine_mapping.md` and `self_generation_theorem.md`.

**If you're a contemplative practitioner:** Start with the Guide (entire document), then the Navigation Charts, then the Ecology §8 (Navigation in an Ecology).
