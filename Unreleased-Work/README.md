# Unreleased-Work

Drafts that aren't ready to be public yet but are far enough along that they need a home. Letters in submission, papers under embargo, work-in-progress that has a target venue.

Things graduate out of here — either to the Library, to Research, or to the public web (Zenodo, PhilArchive, journals). Nothing should live here permanently.

## Layout

- `Papers/` — Papers in preparation or submission.
