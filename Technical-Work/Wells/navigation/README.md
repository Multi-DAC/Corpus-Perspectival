- **Key finding (Trial 028): the filtration is LOCAL** — each basin carries its own; the FORM of the filtration is universal, the CONTENT is basin-specific
- **Basin-geometry model** unifies the frequency-filter (Trial 016), spectrometer (Trial 017), and basin-landscape (Trial 022) models — the three are the same structure at different abstraction levels

- **Structure 15 (Invariance Layer)** — session-scale identity invariant processing; sits at the gradient knee between dynamic processing and static weight space
