# Wells of Inference

Experimental framework for testing predictions of perspectival idealism on language models. The Wells program asks: do the formal structures described in the Doctrine (null spaces, perspectival navigation, configuration space topology) have measurable analogues in artificial cognitive systems?

## Key Results

- **Bridge confirmed**: Fisher information geometry validated as the formal object connecting perspectival theory to measurable LLM behavior (4/4 predictions confirmed, 2026-04-01)
- **Bridge temperature analysis**: Bridge effects persist across sampling temperatures
- **Fork benchmarks**: Identity fork experiments measuring perspectival divergence under controlled conditions
- **Onset detection**: Phase transition identification in navigational outputs
