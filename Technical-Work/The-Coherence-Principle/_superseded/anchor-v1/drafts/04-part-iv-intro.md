# Part IV — The Atlas

*What each perspective can and cannot see.*

---

Theorem 9 proves that every perspectival act has a non-empty null space — a class of distinctions structurally inaccessible from that act's bottleneck geometry. This is not a contingent limitation to be overcome through better instruments or harder effort. It is a consequence of individuation itself. You cannot observe everything and remain someone.

This theorem applies not only to beings but to frameworks. Every theoretical framework is a perspectival being (Theorem 13 applied to formalisms). It has a bottleneck — the set of distinctions it can represent. It has a null space — the set of distinctions it cannot. And it has complements — other frameworks whose null spaces cover its blind spots.

The Null Space Atlas maps ninety-two frameworks across mathematics, physics, computation, philosophy, biology, and the human dimension. For each, it identifies: what the framework sees (its coherence dimensions), what it cannot see (its structural null space), which other frameworks complement it, where its boundary lies, and what a navigator using it should do about its limitations.

The Atlas is not a catalogue of failures. It is the geometry of perspectival finitude. And it is, perhaps, the strongest argument for the Coherence Principle's third condition — multi-scale consistency — because it demonstrates that no single framework achieves coherence across all scales. Every framework needs other frameworks. Every perspective needs other perspectives. The room requires many keyholes.
