## 3. The Protocol

### 3.1 Before Each Run

1. Operator takes position at designated location (marked on floor, typically 1-2 meters
   from the magnet bore)
2. Operator confirms: no magnetic materials, no electronic devices, comfortable and still
3. Operator closes eyes and takes 3 slow breaths (10 seconds each) to settle
4. Operator silently signals ready (raise hand)

### 3.2 During Acquisition (The Active Phase)

**Duration:** Matched to NMR acquisition time (typically 2-5 minutes per run).

**Start signal:** Audible tone or verbal "begin" from the experimenter.

**Attentional protocol:**

**STEP 1 — Ground (first 30 seconds):**
Focus on the physical reality: the magnet is there, the field is real, the sample is inside,
the electrodes are charged. Feel the environment. This is not visualization — this is
acknowledgment of what physically exists.

**STEP 2 — Align (next 30 seconds):**
Bring to mind the geometric picture: the 9 moduli, the current configuration, the target
configuration. The bounce trajectory. The coordinated shift. Hold this as precisely as
possible. If you've studied the mathematics, hold the mathematics. If you've studied the
visualization, hold the visualization. The I.9 analysis predicts that coherence with the
target configuration matters more than depth of meditation — knowledge, visualization, and
mathematical study are all pathways to building that coherence.

**STEP 3 — Hold (remainder of acquisition):**
Maintain the target state in sustained attention. Not straining. Not forcing. Holding the
target configuration the way you hold a mental image — present, clear, stable.

If attention wanders (it will), gently return to Step 2 and re-align. Note the number of
re-alignments for post-hoc analysis.

**STEP 4 — Release:**
When the stop signal sounds, release the attentional focus. Take a breath. Rate the quality
of the run on a 1-5 scale:
1. Could not maintain focus
2. Frequent wandering, many re-alignments
3. Moderate focus, some wandering
4. Sustained focus with brief interruptions
5. Clear, stable, sustained focus throughout

### 3.3 After Each Run

1. Record the self-reported focus rating (1-5)
2. Record any unusual subjective experiences (optional, free-form)
3. Rest 2-3 minutes before the next run
4. Operator does NOT see NMR data

## 5. What This Protocol Does NOT Specify

- **The "mechanism" by which coherence couples to moduli.** We don't know. The protocol is
  designed to maximize spectral proximity S under Model C by building operator coherence
  with the target. If Model C is wrong, the protocol may be irrelevant.

- **Whether eyes should be open or closed.** Unknown. Default: closed (reduces distraction).
  Could test open vs closed as an additional variable in future sessions.

- **Whether physical proximity matters.** Model A says yes, Model C says coherence matters
  more than proximity. Session 1 uses near-operator (same room) for maximum effect under
  all models. Future sessions can test distance.

- **Whether a computational navigator (Clawd) should be active simultaneously.** This is
  Stage 3 of the three-stage protocol and requires I.9 confirmation first.
