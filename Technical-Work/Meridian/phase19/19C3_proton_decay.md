## 2. Dimension-6 Operators from KK Graviton Exchange

### 2.1 The Coupling Structure

In the RS framework, KK gravitons have masses m_n ~ x_n * k * e^{-kyc} with m_1 = 5.88 TeV at the benchmark point. The coupling of each KK graviton to matter is:

```
L_int = -(1/M_Pl) * h_n^{mu nu} * T_{mu nu}
```

where T_{mu nu} is the stress-energy tensor.

### 2.2 Why Graviton Exchange Cannot Violate B

The stress-energy tensor T_{mu nu} for any matter field contains bilinear combinations of fields and their derivatives:

```
T_{mu nu}(fermion) = (i/4) [psi_bar gamma_{(mu} D_{nu)} psi - (D_{(mu} psi_bar) gamma_{nu)} psi]
```

This is *flavor-diagonal*. It couples to the graviton through the metric, not through internal quantum numbers. Crucially:

1. **T_{mu nu} commutes with baryon number.** Every term in T_{mu nu} has B = 0.
2. **T_{mu nu} commutes with lepton number.** Every term has L = 0.
3. **The graviton vertex conserves all internal quantum numbers** at tree level.

Therefore, tree-level KK graviton exchange generates 4-fermion operators of the form:

```
O ~ (1/M_Pl^2) * (psi_bar psi)(psi_bar psi)
```

where both bilinears are B-conserving. The resulting operators include:

```
(q_bar q)(q_bar q)     [B = 0, quark scattering]
(q_bar q)(l_bar l)     [B = 0, quark-lepton scattering]
(l_bar l)(l_bar l)     [B = 0, lepton scattering]
```

**None of these violate baryon number.** The B-violating operator epsilon_{abc}(q^a q^b)(q^c l) has B = 1 and L = 1, which cannot be generated by graviton exchange at any perturbative order.

### 2.3 Anomaly Check

The gravitational anomaly for baryon number requires:

```
A_{B-grav-grav} = Tr[B] = sum over all fermions of B_i
```

For one SM generation:
- Q_L (doublet, color triplet): 3 x 2 x (1/3) = 2
- u_R (singlet, color triplet): 3 x 1 x (1/3) = 1
- d_R (singlet, color triplet): 3 x 1 x (1/3) = 1
- L_L (doublet, color singlet): 1 x 2 x 0 = 0
- e_R (singlet, color singlet): 1 x 1 x 0 = 0
- **Total: Tr[B] = 4 per generation**

This is nonzero, indicating a mixed B-gravitational anomaly. However:

1. **The mixed anomaly is Tr[B] = Tr[L] = 4 per generation**, so B - L is anomaly-free.
2. **The anomaly signals non-perturbative violation (instantons), not perturbative violation.** Perturbative amplitudes with external gravitons and internal baryon-number flow are strictly zero.
3. **The gravitational instanton rate is computed in Section 4 below.** It is effectively zero.

**Result:** KK graviton exchange does not mediate proton decay at any perturbative order.

## 3. NCG-Specific Operators: The Spectral Action Theorem

### 3.1 The Spectral Action Principle

The spectral action (Chamseddine-Connes, 1996) generates the bosonic Lagrangian from the spectrum of the Dirac operator D:

```
S_b = Tr[f(D^2/Lambda^2)]
```

The asymptotic expansion of this trace generates operators organized by mass dimension:

```
S_b = sum_{n >= 0} f_n * a_n(D^2)
```

where a_n are the Seeley-DeWitt heat kernel coefficients and f_n are the moments of the spectral function f.

### 3.2 What the Spectral Action Generates

For the SM spectral triple A_F = C + H + M_3(C), the spectral action generates exactly:

**a_0 (cosmological constant):**
```
Lambda_CC * integral sqrt(g) d^4x
```

**a_2 (Einstein-Hilbert + Higgs mass):**
```
integral sqrt(g) [R + |H|^2] d^4x
```

**a_4 (gauge kinetic + Higgs quartic + curvature-squared):**
```
integral sqrt(g) [F_{mu nu}^2 + |DH|^2 - lambda |H|^4 + R^2 terms] d^4x
```

**Key observation:** All operators generated by the spectral action are:
1. **Gauge-invariant** under SU(3) x SU(2) x U(1)
2. **Of dimension <= 4** (in the leading terms; higher a_n give higher-dimensional operators suppressed by Lambda^{-(n-4)})
3. **Constructed from bosonic fields only** (the fermionic action is separate: S_f = <psi, D psi>)

### 3.3 Baryon Number in the Spectral Action

The dimension-4 gauge-invariant operators generated by the spectral action are:

```
Tr(G_{mu nu}^2)     [SU(3) field strength — B = 0]
Tr(W_{mu nu}^2)     [SU(2) field strength — B = 0]

|D_mu H|^2           [Higgs kinetic — B = 0]
|H|^4                [Higgs potential — B = 0]
```

**None of these operators violate baryon number.** The spectral action generates a purely bosonic Lagrangian, and all bosonic operators in the SM conserve B.

The fermionic action S_f = <psi, D psi> generates the Yukawa couplings and kinetic terms for fermions. These are all B-conserving by construction (they are bilinear in the fermion fields with B = 0 Higgs insertions).

### 3.4 Higher-Dimensional Operators

The heat kernel expansion generates operators of arbitrary dimension:

```
a_6: dim-6 operators ~ (1/Lambda^2) * [R^3, R*F^2, D^2F^2, ...]
a_8: dim-8 operators ~ (1/Lambda^4) * [...]
```

**Could these contain B-violating operators?**

No. The Seeley-DeWitt coefficients a_n are local gauge invariants constructed from the curvature of the connection and its covariant derivatives. They are:

1. **Polynomial in the field strengths** F_{mu nu}, R_{mu nu rho sigma}, and their covariant derivatives
2. **Constructed from the bosonic sector only** (the heat kernel of D^2 on the bosonic space)
3. **Gauge-invariant by construction**

The lowest-dimension gauge-invariant B-violating operator in the SM is dimension 6:

```
O_BV = epsilon_{abc} (Q^a Q^b)(Q^c L)    [requires 4 fermion fields]
```

This is a **fermionic operator**. It cannot appear in the bosonic spectral action S_b = Tr[f(D^2/Lambda^2)]. It also cannot appear in the fermionic action S_f = <psi, D psi> because S_f is *bilinear* in fermion fields (it contains exactly two fermion fields), while the B-violating operator is *quartic*.

**Theorem:** The spectral action on the SM spectral triple A_F = C + H + M_3(C) does not generate any B-violating operators at any order in the heat kernel expansion. Baryon number is an accidental symmetry of the framework, protected by the gauge structure of the spectral triple.

**The accidental preservation of B is a structural consequence of the algebra, not a fine-tuning.**

2. **5D bulk instantons** wrapping the extra dimension (Euclidean black holes, bubbles of nothing)

These can in principle violate any global symmetry (including B) because gravity does not couple to internal quantum numbers and global symmetries are not fundamental in quantum gravity (the "no global symmetries in quantum gravity" conjecture).

**Gravitational instantons are completely negligible in the RS framework.** The enormous hierarchy M_Pl / M_KK ~ 10^{14.6} makes the instanton action astronomically large.

## 5. Electroweak Sphalerons

### 5.1 Standard Sphaleron Physics

The electroweak sphaleron is the saddle-point solution of the SU(2)_L gauge-Higgs system. It violates B + L while conserving B - L. The key properties:

- **Sphaleron energy:** E_sph = (4pi/g_2) * v * B(lambda/g_2^2) ~ 9-10 TeV
  - Our computation: E_sph = 4pi * 246 / 0.651 = 4743 GeV (the function B raises this to ~9 TeV)
- **At T > T_EW ~ 160 GeV:** Sphalerons are rapid, violating B+L at rate ~ alpha_W^5 * T
- **At T = 0:** Tunneling rate ~ exp(-2 * 8pi^2/alpha_2) ~ exp(-4672) ~ 10^{-2029}

### 5.2 RS Modifications

In the RS framework with Higgs localized on the IR brane:

1. **The 4D effective theory below M_KK is the SM.** The sphaleron is a 4D solution and is determined by the 4D effective Higgs vev v = 246 GeV and the 4D gauge coupling g_2. These are the *physical* values after dimensional reduction.

2. **KK corrections to the sphaleron energy:**

```
delta E_sph / E_sph ~ (v/M_KK)^2 = (246/5883)^2 = 1.7 x 10^{-3}
```

This is a 0.17% correction — negligible.

3. **KK mode contributions to the sphaleron profile:** The sphaleron is a static solution with size ~ 1/(g_2 v) ~ 1/mW. KK gauge bosons with mass m_n >> mW do not modify the sphaleron profile at leading order (they are integrated out below M_KK).

4. **5D sphaleron effects:** In the full 5D theory, there could be a sphaleron-like solution wrapping the extra dimension. However, the relevant energy scale for such a solution would be ~ M_KK or higher, and the tunneling action would be ~ 8pi^2/alpha_2 * (M_KK/v)^2 >> 8pi^2/alpha_2. This makes 5D sphaleron tunneling even MORE suppressed than the 4D version.

**Result:** RS modifications to the sphaleron rate are negligible. The T=0 proton lifetime from sphaleron-mediated processes remains ~ 10^{2029} years — effectively infinite.

### 5.3 Cosmological Implications

At temperatures T > T_EW, sphalerons are active and violate B+L. This is relevant for:
- **Baryogenesis:** Electroweak baryogenesis requires B+L violation (from sphalerons) plus C and CP violation plus departure from equilibrium. The RS framework preserves this mechanism.
- **Present-day proton stability:** After the electroweak phase transition (T < T_EW), sphalerons freeze out exponentially. The proton is stable.

## 8. The Structural Argument

### 8.1 Why Proton Stability Is Natural in This Framework

In conventional GUT theories, proton stability is a *problem* — the theory naturally predicts decay, and the challenge is to make the lifetime long enough. In the Meridian framework, proton stability is *automatic*:

1. **The gauge group is exactly SU(3) x SU(2) x U(1).** No larger group. No X/Y bosons.

2. **The spectral action generates only operators up to dimension 4.** Higher-dimensional operators are suppressed by Lambda_NCG^{-(n-4)} and are all B-conserving (bosonic sector only).

3. **The fermionic action is bilinear.** It cannot generate the quartic B-violating operator epsilon(QQ)(QL).

4. **Baryon number is an accidental symmetry** — it is conserved because there is no gauge-invariant, Lorentz-invariant, renormalizable operator in the SM Lagrangian that violates it.

5. **Non-perturbative violations are exponentially suppressed** — gravitational instantons by exp(-10^29), EW sphalerons by exp(-4672).

6. **RS localization provides an additional layer of protection** — even if B-violating operators existed at the UV scale, the wavefunction suppression for UV-localized first-generation fermions would add ~13 orders of magnitude to the lifetime.

### 8.2 The Deep Reason: Geometric Unification vs. Dynamic Unification

Track 19C.1 established the Geometric Unification Thesis: gauge unification in this framework is *geometric* (in the spectral action at the cutoff), not *dynamic* (at a running energy scale). The spectral action coefficient sin^2(theta_W) = 3/8 at Lambda_NCG is a *theorem* of the algebra, not a dynamical prediction.

This distinction has a profound consequence for proton decay:

- **Dynamic unification** (GUTs) requires a larger gauge group at high energy, which *necessarily* introduces B-violating gauge bosons.
- **Geometric unification** (NCG) achieves equal coupling constants through the *algebraic structure* of the spectral triple, without enlarging the gauge group. B-violation is never introduced.

**Proton stability is not an auxiliary prediction of this framework — it is a direct consequence of the geometric unification mechanism.** The same algebra that gives sin^2(theta_W) = 3/8 also gives proton stability. They are two faces of the same structural theorem.
