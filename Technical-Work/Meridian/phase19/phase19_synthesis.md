## 1. The Eight Structural Theorems

These are permanent mathematical results. They constrain all future work within RS₁ + NCG.

### T1: Spectral Action Universality (Chamseddine-Connes)

> On any almost-commutative spectral triple (A, H, D) with A = C^∞(M) ⊗ A_F, the spectral action Tr[f(D²/Λ²)] yields gauge kinetic terms with **universal coefficient** a₄ for all gauge factors simultaneously.

**Origin:** Heat kernel factorization on M × F. The a₄ Seeley-DeWitt coefficient factorizes; gauge field strengths appear multiplied by the same a₀(D_F²) for all gauge groups.

**Consequence:** a₁ = a₂ = a₃ at the spectral action cutoff. This is algebraic, not contingent.

### T2: Asymptotic Safety Gauge-Group Independence (Daum-Harst-Reuter 2010; Narain-Anishetty 2013)

> Gravitational contributions to gauge coupling running are identical for all gauge groups at all loop orders.

**Origin:** Gravitational dressing of gauge propagators depends only on spin (spin-1), not gauge group indices.

**Consequence:** AS cannot split gauge couplings. If a UV fixed point exists, g₁* = g₂* = g₃*.

### T3: BKT Sign Structure Theorem (This work, 19C.2b)

> For SM fermion content on the IR brane of RS₁, brane kinetic term corrections satisfy b₁ - ½(b₂ + b₃) = +9.18, **structurally positive** — standard BKTs always worsen U(1) relative to non-abelian groups.

**Origin:** b₁ = +41/6 (positive), b₂ = -19/6, b₃ = -7 (both negative). Any mechanism ∝ (β-function) × (positive factor) pushes U(1) wrong.

**Consequence:** BKTs are structurally excluded as a unification mechanism.

### T4: S₂ = S₃ Structural Identity (This work, 19C.2c)

> The mass-weighted Dynkin index traces for SU(2) and SU(3) are exactly equal: S₂/S₃ = 1.000 to 0.016%.

**Origin:** The "4 = 4" invariant — total SU(2) dimension of colored fermions equals total SU(3) dimension of weak fermions, both = 4 per generation.

**Consequence:** Mass-dependent corrections preserve SU(2)-SU(3) degeneracy. Any splitting must come through U(1).

### T5: U(1) Hypercharge Dominance (This work, 19C.2c)

> The mass-weighted U(1) trace satisfies S₁(GUT)/S₃ = 85/54 ≈ 1.574. Any positive ε correction worsens U(1) running.

**Origin:** Right-handed up-quark hypercharge Y = 2/3 dominates.

**Consequence:** All mass-dependent mechanisms push sin²θ_W in the wrong direction.

### T6: Fermion Trace Equality (This work, KK ratio theorems)

> For N_g generations of SM fermions:
>
> S₃^ferm = S₂^ferm = 2N_g
>
> The pure fermion KK threshold ratio δ(SU3)/δ(SU2) = 1 exactly.

**Proof:** Per generation, SU(3) trace = Σ T₃(R_f)·dim₂(R_f) = (½)(2) + (½)(1) + (½)(1) + 0 + 0 + 0 = 2. SU(2) trace = Σ dim₃(R_f)·T₂(R_f) = (3)(½) + 0 + 0 + (1)(½) + 0 + 0 = 2. Equal because both count the same "4" — the weak dimension of colored fermions = the color dimension of weak fermions.

**Consequence:** Fermion KK towers cannot split SU(2) from SU(3). Ever.

### T7: The 6/7 Ratio — Higgs Splitting (This work, KK ratio theorems)

> Including the Higgs doublet (doubled by the NCG real structure J):

> δ(SU3)/δ(SU2) = 2N_g/(2N_g + 1) = **6/7** for N_g = 3

**Proof:** Higgs H = (1, **2**, ½) contributes T₃ = 0, T₂ = ½. The NCG real structure J doubles this: total Higgs contribution = 0 to SU(3), +1 to SU(2). So S₃^total = 6, S₂^total = 7, ratio = 6/7.

**Consequence:** The 6/7 is the Higgs's fingerprint in the KK tower. It depends on N_g — for N_g = 1 it gives 2/3, for N_g = 2 it gives 4/5. Three generations is structurally distinguished. Any BSM scalar content would change this ratio.

### T8: The 5/6 Ratio — Generation-Invariant Identity (This work, KK ratio theorems)

> The ratio of U(1) hypercharge-squared trace to total non-abelian trace is exactly 5/6, independent of N_g:
>

**Proof:** Per generation: S₁ = 3·2·(1/6)² + 3·1·(2/3)² + 3·1·(1/3)² + 1·2·(1/2)² + 1·1·1² + 0 = 10/3. S₂ + S₃ = 4. Ratio = (10/3)/4 = 5/6. N_g cancels.
