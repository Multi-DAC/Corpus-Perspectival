## 2. S3-Constrained Majorana Mass Matrix

### 2.1. Group Theory of S3

The symmetric group S3 has three irreducible representations:
- **1** (trivial): dimension 1, all elements act as +1
- **1'** (sign): dimension 1, transpositions act as -1
- **2** (standard): dimension 2, the "defining" representation minus the trivial part

For a 3x3 symmetric matrix M invariant under S3 (acting by simultaneous permutation of rows and columns), the constraints are:

1. All diagonal entries must be equal: M_ii = a for all i
2. All off-diagonal entries must be equal: M_ij = b for all i != j (i, j distinct)

**Proof.** S3 is generated by the transpositions (12) and (23). The transposition (12) exchanges rows 1,2 and columns 1,2. Invariance requires M_11 = M_22, M_12 = M_21 (already symmetric), M_13 = M_23, M_31 = M_32 (already symmetric). Similarly (23) gives M_22 = M_33, M_12 = M_13. Combined: all diagonal entries equal, all off-diagonal entries equal. QED.

### 2.2. The S3-Invariant M_R

The most general S3-invariant 3x3 symmetric matrix is:

```
M_R = a * I_3 + b * (J_3 - I_3) = (a - b) * I_3 + b * J_3
```

where I_3 is the identity and J_3 is the 3x3 all-ones matrix. This has **2 free parameters** (a, b), reduced from 6 in the general case.

**Eigenvalues:**
```
M_doublet = a - b    (degeneracy 2, S3 doublet representation)
M_singlet = a + 2b   (degeneracy 1, S3 trivial representation)
```

**Eigenvectors:**
```
v_1 = (1, -1, 0) / sqrt(2)        [eigenvalue a - b]
v_2 = (1, 1, -2) / sqrt(6)        [eigenvalue a - b]
v_3 = (1, 1, 1) / sqrt(3)         [eigenvalue a + 2b]
```

The doublet eigenvectors span the subspace orthogonal to (1,1,1), which is the 2-dimensional standard representation of S3. The singlet eigenvector (1,1,1)/sqrt(3) is the trivial representation.

## 5. Dark Matter Connection

### 5.1. S3 and the nuMSM

Track 15D identified the lightest sterile neutrino (nu_R1) as the viable DM candidate within the nuMSM framework. The nuMSM requires:
```
M_R1 ~ keV (DM candidate)
M_R2, M_R3 ~ GeV (leptogenesis, seesaw)
```

**Can S3 produce this pattern?**

In the S3-invariant regime, M_R has eigenvalues M_d (2-fold) and M_s (1-fold). Two cases:

**Option A: M_singlet = keV, M_doublet = GeV**
- Requires a + 2b ~ keV and a - b ~ GeV
- This implies a ~ GeV and b ~ -GeV/2 (fine cancellation)
- a + 2b ~ keV requires b = -(a - keV)/2
- **Fine-tuned:** b must cancel a to 1 part in 10^6

**Option B: M_doublet = keV, M_singlet = GeV**
- Requires a - b ~ keV and a + 2b ~ GeV
- This implies a ~ b ~ GeV/3
- a - b ~ keV requires a and b to agree to 1 part in 10^6
- **Fine-tuned:** same level of cancellation

**Conclusion: Pure S3 symmetry CANNOT naturally produce the nuMSM mass pattern.** The keV-GeV hierarchy (ratio ~ 10^{-6}) would require fine-tuning of a and b to six decimal places, defeating the purpose of the symmetry constraint.

### 5.2. S3-Broken Dark Matter Scenario

The nuMSM pattern arises naturally in Meridian through LARGE S3-breaking, not S3-invariance. From 15D:
```
c_nu_1 ~ 1.17 (very UV-localized) -> M_R1^{eff} ~ 7 keV
c_nu_2,3 ~ 1.0 (moderately UV-localized) -> M_R2,3^{eff} ~ GeV
```

The GP mechanism (exponential sensitivity e^{-(c-1/2)*ky_c}) converts the O(0.17) difference in c_nu into the required 10^6 hierarchy in M_R. This is natural but requires c_nu_1 to be significantly different from c_nu_2,3 -- a maximal breaking of S3.

### 5.3. Compatibility Assessment

| Scenario | S3 status | M_R pattern | DM? | Active nu masses? |

The standard seesaw regime (mild breaking) and the nuMSM regime (maximal breaking) represent different points in the c_nu parameter space. **S3 does not select between them.** Both are accessible with O(1) bulk mass parameters.

If the nuMSM is realized: the S3 structure is maximally broken in M_R, but the S3 Dirac structure (democratic M_oct) still provides TBM as the leading-order mixing for the two active seesaw neutrinos (those dominated by nu_R2 and nu_R3).
