C_KK is NOT a universal FRW kinematic constant.

  C_KK = (1+q_0)^2 Omega_DE / [8 (1-q_0)^2 zeta_0]     (*)  

This formula arises from:
  1. The cuscuton constraint: V'_eff = -3H mu^2 sign(phi_dot)
  2. The NMC effective mass: V''_eff = 2(1-q_0) H_0^2
  3. The dark energy condition: mu^2 = Omega_DE M_Pl H_0 / phi_0
  4. The KK reduction: phi_0^2 = 3 zeta_0 M_Pl^2

========================================================================
CROSS-CHECK: DOES THE FORMULA REPRODUCE KNOWN LIMITS?
========================================================================

1. zeta_0 -> 0 (no NMC): C_KK -> infinity.
   This is correct: without NMC, V''_eff = 0 for linear potential,
   and the formula (78-80) breaks down. The cuscuton has no effective
   mass and phi_dot is undetermined.

2. q_0 -> -1 (pure de Sitter): 1+q_0 -> 0, C_KK -> 0.
   Correct: in pure de Sitter, H_dot = 0, so phi_dot = 0,
   K_eff = 0, and w = -1 exactly.

3. Omega_DE -> 1 (DE dominated): q_0 -> -1, C_KK -> 0.
   Same as above — DE domination means H_dot -> 0.

4. Large zeta_0: C_KK -> 0.
   Correct: strong NMC creates large V''_eff, which suppresses phi_dot
   and therefore K_eff, pushing w closer to -1.

All limits are physically correct.
