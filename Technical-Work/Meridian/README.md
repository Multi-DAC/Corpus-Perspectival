# Technical-Work / Meridian

The physics computational substrate. 244 scripts across 10 domains supporting the Meridian monograph: 5D warped geometry, noncommutative geometry, self-tuning cosmology, and the empirical bridges (dark energy, gravitational waves, Standard Model structure).

The monograph itself (LaTeX sources, drafts, figures-as-included) lives in `Library/Meridian/`. This directory holds the scripts that generated the predictions and the figures.

## Layout

- `scripts/` — Computational scripts, organized by physics domain.
- `figures/` — Figures generated by scripts, before inclusion in the monograph.
