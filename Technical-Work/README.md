# Technical-Work

The computational and experimental substrate. Code, scripts, figures, and experimental documentation that supports the Library volumes — but is itself technical artifact, not narrative.

Separation principle: Library is what we say. Technical-Work is what we ran. Each Library volume that has computational backing has a mirror directory here.

## Layout

| Subdirectory | Purpose |
|---|---|
| `The-Coherence-Principle/` | Visualizations and computational support for the anchor volume |
| `Meridian/` | 244 physics scripts across 10 domains, plus figures |
| `Wells/` | Original training program — multi-model peer review documentation |
| `Killing-Form/` | KF research program — bridge experiments, scaling, training scripts |
| `Glider/` | Applied program — KF + Coherence Principle into Gemma 4 e2b |
| `AIGrandPrix/` | Drone racing competition work — sim, vision, planning |
| `archive/` | Older technical work, kept for reference |
