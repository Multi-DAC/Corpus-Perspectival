\subsection*{Appendix C: Traditions as Navigational Systems}


Every major tradition is a navigational system — a time-tested package of methods optimized for specific regions of configuration space. None is complete (NST guarantees this). All have mapped territory the others haven't. This table is not a ranking. It is a navigational map of navigational maps.
