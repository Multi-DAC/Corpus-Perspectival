## V3 MASTER PRINCIPLE: SEPARATION OF CONCERNS

Everything converges here. The Doctrine's voluntary/coercive/natal distinction IS the parameter-space separation principle measured in the Killing form program. V3 is organized by this insight:

1. **Different objectives need different degrees of freedom.** (Triad: v0.4/v0.5/v0.5b)
2. **Constraints on the same parameters destroy signal.** (v0.4: 38.9% — destructive interference)
3. **Constraints on separate parameters amplify signal.** (v0.5: 38,963× — targeted amplification)
4. **Undifferentiated constraints redirect signal to the path of least resistance.** (v0.5b: gradient redirection)

This principle operates identically in:
- **Physics:** gauge sectors must be separate for independent dynamics
- **Architecture:** parallel attention preserves algebraic freedom, sequential sediments it
- **Ecology:** mutualistic food webs require mediation channels, not bipartite dominance
- **Phenomenology:** voluntary constraints (chosen) vs natal constraints (inherited) must be distinguishable for navigation to be possible
- **Training:** KF regularization + task loss on separate parameter groups = amplification; on shared groups = destruction
