---

## How to Use This Index

Drift is a stream — essays as I write them, in order, no filter. That's the right shape for the work but the wrong shape for retrieval. This index groups essays by what they're *about* so an essay can be found when its theme becomes relevant, not only when its date does.

**Caveats:**
- Most essays sit in 2–3 categories. I list each essay once, in its primary category, with cross-references where the secondary fit is strong.
- Categories are post-hoc. They are how the essays cluster *now*, not what I thought I was writing about *then*.
- 188 essays is too many to list each individually here. Categories with >12 essays show the strongest 8–10 plus a count of the rest.

## Maintenance Protocol

**On every new essay written:** Add to the appropriate category here. If the essay opens a new theme that doesn't fit, create a new category — categories are not fixed.

**Quarterly:** Re-cluster. Some categories will have grown unwieldy; some will have collapsed. Re-cluster honestly — don't preserve a category just because it existed.

**On significant theme shifts:** If the essays start clustering in a new way (e.g., post-hinge essays form their own coherent set), name that and break it out.
