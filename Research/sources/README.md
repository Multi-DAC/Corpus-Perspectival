# Sources Register

*Rolling list of all papers, articles, talks, and URLs read, cited, or discussed in the Coherence Principle program. Established 2026-04-19 (Day 78) after discovering that in-thread URL shares were being lost to truncated log summaries.*

---

## Operational rule

**Every URL, PDF, or paper shared or discussed gets an entry in this register the same session вЂ” not "later."** Same discipline as the falsification-clause requirement for new bridges: the entry is load-bearing for future work, so it cannot be deferred.

If a source is discussed but no entry exists, either (a) create it immediately, or (b) flag in the working document that the entry is missing and must be made before publication.

---

## Entry format

Each source lives at `sources/YYYY-MM-DD-slug.md` where `YYYY-MM-DD` is the date first encountered (not the date the source was published) and `slug` is a short human-readable identifier. Frontmatter fields:

- `url` вЂ” primary URL
- `archive` вЂ” archive.org mirror URL (added when possible)
- `title` вЂ” exact title
- `author` вЂ” author(s)
- `venue` вЂ” publisher / journal / outlet
- `published` вЂ” publication date
- `accessed` вЂ” date first fetched/read by us
- `discussed` вЂ” date(s) first discussed in the program
- `tags` вЂ” comma-separated: which paper/volume/bridge it's relevant to
- `status` вЂ” `read-in-full`, `read-skim`, `pending-read`, `discussed-not-read`

Body: (~200-400w)

- **What it argues** вЂ” charitable summary
- **Where we agree** вЂ” the insights we incorporate
- **Where we diverge** вЂ” what we dispute
- **Connection to our program** вЂ” which papers/volumes/bridges it bears on
- **Quote-pulls** вЂ” direct text we may cite, with page or paragraph refs
