### Phase 1 protocol design — DIY multi-frequency EM + light platform

**Build target.** Controllable waveform generator + power amplifier + Helmholtz coil pair + Hall-sensor field measurement + optional integrated LED array, capable of:
- Frequency sweep 0.1 Hz to ~20 Hz (covering delta through beta; future MHz-range capability if needed)
- Variable amplitude up to ~10 mT (Akdag's amplitude; consumer-device-comparable)
- Variable waveform shape (sine / sawtooth / square)
- Optional 660 nm + 850 nm LEDs gated from same waveform generator

**Cost:** ~$200-300 in parts. Build time: 1-2 weekends if soldering-comfortable; longer if learning.

**Architecture:**
```
[Microcontroller + AD9833 DDS] → [Power amplifier] → [Helmholtz coil pair]
                  ↓                                              ↓
            [Hall-sensor feedback ←———————————————]
                  ↓
            [LED driver + 660/850nm array, optional]
```

**Parts list (cheap-start):**
- Subsystem 1 (waveform gen, ~$30-50): Teensy 4.0 + AD9833 DDS module
- Subsystem 2 (amplifier, ~$30-80): op-amp + power transistor stage (DC-coupled, delta-range performance) preferred; alternative: TPA3116 + DC-coupling mod
- Subsystem 3 (coils, ~$30-50): Helmholtz pair, ~150 turns AWG 20 magnet wire, 30 cm diameter, 15 cm separation
- Subsystem 4 (field measurement, ~$20-40): Hall sensor (HMC5883L or DRV5055) — NON-NEGOTIABLE for calibrated experiments
- Subsystem 5 (LEDs, optional, ~$25-50): 660+850 nm LEDs + constant-current driver + PWM gate
- Power + housing + misc: ~$50-80

**Construction stages:**
1. Waveform generator alone, oscilloscope-verified clean sine at target frequencies
2. Add amplifier, drive small test coil, measure with Hall sensor (FAR from body); calibrate measured-vs-calculated field
3. Scale to Helmholtz pair; map field volume; identify where 10 mT region is and where it falls below 1 mT
4. Calibration + safety verification (thermal, continuous-operation, clean-shutdown)
5. Add optional LED array; verify pulse-matched to EM
6. Body exposure begins ONLY after stages 1-5 complete

**Pre-body-exposure safety checklist:**
- Field measured at exposure location, confirmed ≤ 10 mT
- Field falloff mapped
- No-implants check (no pacemaker / ICD / DBS / cochlear / programmable shunt; also: nobody else in the room with implants)
- No-cardiac-arrhythmia / no-seizure-history check
- Power supply fused; mains-isolation verified; no exposed conductors at body-contact distance
- Emergency-stop within reach
- Continuous-operation tested for full intended duration without thermal issues
- First exposure: 5 min instead of 15; monitor for unusual sensation; abort and reassess if anything weird
