# Research

In-progress notes, reading register, and a curated index of all the research we cite, compile, or build on across the Corpus.

Two functions:

1. **Working notes** — Per-domain directories hold drafts, sketches, and the "I'm thinking about this" layer that hasn't yet earned a Library volume slot.
2. **Reading register** — A document linking out to all the external research we use, organized by domain. This is the citation substrate. Where the Library draws from.

## Layout

| Subdirectory | Purpose |
|---|---|
| `The-Coherence-Principle/` | Notes feeding the anchor volume |
| `Meridian/` | Physics research notes |
| `The-Killing-Form/` | KF program research notes |
| `The-Living-Architecture/` | Living architecture / dynamic systems notes |
| `The-Coherent-Body/` | Body/biology application of the Principle (the "body" node of the body/mind/business triad seeded by the friend's conversation) |
| `Misc/` | Cross-cutting or unfiled |

The reading-register document lives at the top of this directory once compiled.
