## How This Works

- **Source material stays in place.** The `corpus/`, `meridian/`, `experiments/`, `drift/`, `paper/` directories are the lab. They don't move.
- **Book directories are compilation workspaces.** Each `books/<volume>/` directory contains drafts, outlines, and a README mapping sources to chapters.
- **Drafts go in `drafts/`.** Working text that will become the published volume.
- **Each README is the book's blueprint.** It says what the book contains, where the source material lives, what's been drafted, and what remains.

## The Principle Applied to Its Own Library

Each volume operates on separate degrees of freedom. No volume competes with another. Each volume's null space is filled by another volume's perspective. The library grows through exploration, not through expanding any single volume. Separation, measurement, multi-scale consistency, dynamic maintenance.
