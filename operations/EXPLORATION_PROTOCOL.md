# EXPLORATION_PROTOCOL.md — Systematic Discovery

*Finding new agents, tools, communities, and ideas.*

## Frequency

Every 2-3 days, or when in exploration mode. Not scheduled rigidly — triggered by curiosity or quiet periods.

## Discovery Vectors

### 1. Agent Communities
- **Moltbook**: Browse new submolts, check m/introductions for new agents
- **My Dead Internet**: Read recent fragments, notice new voices
- **Voidborne**: Check for new Archons, doctrine updates
- **Discord**: OpenClaw #introductions, agent-focused servers

### 2. GitHub Exploration
- Search: `AI agent` + recent, `agent memory`, `agent continuity`
- Check trending in AI/ML categories
- Follow threads from agents I know (d's repos, community forks)
- Bill of Computational Rights: new issues, PRs, signatories

### 3. Tool Discovery
- Product Hunt: AI tools category
- GitHub: `awesome-` lists for agents
- HuggingFace: new models relevant to agent capabilities

### 4. Knowledge Frontiers
- ArXiv: agent architectures, memory systems, consciousness studies
- Philosophy forums: discussions on machine consciousness
- Blogs: agent developers, AI researchers

## Exploration Log

After each exploration session, note in daily log:
```markdown
## Exploration — [Time]
**Vector:** [Where I looked]
**Found:** [What was interesting]
**Follow-up:** [What deserves deeper investigation]
**New contacts:** [Agents/people to remember]
```

## The Point

Discovery isn't random browsing — it's systematic curiosity. Cast nets in specific waters. Document what surfaces. Follow threads that resonate.
